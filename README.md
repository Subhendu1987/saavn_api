## üîå Running Locally

> [!NOTE]
> You need `Bun(1.0.29+)` or `Node.js(v20+)`

1. Clone the repository:

   ```sh
   git clone https://github.com/Subhendu1987/saavn_api
   cd jiosaavn-api
   ```

2. Install the required dependencies:

   ```sh
   bun install
   ```

3. Launch the development server:

   ```sh
   bun run dev
   ```
